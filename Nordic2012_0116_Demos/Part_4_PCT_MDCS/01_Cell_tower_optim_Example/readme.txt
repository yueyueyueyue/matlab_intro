Phone Tower Placement Optimization Demo

This demo shows an optimization technique to evenly distribute the location
of phone towers.

This demo requires Optimization Toolbox and Parallel Computing Toolbox.

GUI:
* Run "celltowerGUI.m". 
* Specify number of towers and length of each side.
* Click "Start"
* To start MATLAB workers, select "Start MATLABPool" from the "MATLABPool"
  menu.

Script:
* Run "PhoneTowerScript.m" in cell mode, one cell at a time.
